# busylib Docs

`busylib` is a Python library for driving Busy Bar devices over HTTP and WebSocket APIs.

This documentation is intentionally example-first and focuses on real scenarios you can run and adapt.

## Repository Boundaries

The monorepo layout is organized by responsibility:

- `services/`: only product services.
- `shared/`: shared libraries and contracts.
- `playground/`: experimental code and simulator-based validation.

See [Repository Structure](repository-structure.md) for the detailed rules.

## Featured Scenario

- **Now Playing**: render the currently playing track metadata as a live Busy Bar view.
